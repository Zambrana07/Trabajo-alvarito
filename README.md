# TECNOLOTGIA

Sitio web estático listo para GitHub Pages.

## Publicar en GitHub Pages

1. Sube este repositorio a GitHub (rama `Trabajo` o `main`).
2. En el repositorio: **Settings** → **Pages**.
3. En **Source** elige **Deploy from a branch**.
4. En **Branch** selecciona `Trabajo` (o la rama donde esté el código) y carpeta **/ (root)**.
5. Guarda. La página se publicará en `https://tu-usuario.github.io/nombre-repo/`.

La entrada del sitio es el **index.html** en la raíz del proyecto.
